# 🕹️ Jogos R&R - Praticando Lógica de Programação

Este é um projeto desenvolvido como parte do curso de Lógica de Programação da **Alura**. O objetivo principal é aplicar conceitos fundamentais de JavaScript na manipulação do DOM (Document Object Model) e na construção de regras de negócio para uma loja de aluguel de jogos de tabuleiro.

---

## 🚀 Funcionalidades do Projeto

* **Alteração Dinâmica do DOM:** Modificação do título da página para **Jogos R&R** e dos estados dos elementos diretamente pelo JavaScript.
* **Controle de Status de Aluguel:** Lógica para alternar o status dos jogos entre "Disponível" e "Alugado".
* **Botões Interativos:** Modificação dinâmica do texto dos botões (`Alugar` / `Devolver`) e das classes CSS para feedback visual instantâneo ao usuário.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação da página da loja.
* **CSS3:** Estilização moderna e layout responsivo.
* **JavaScript (ES6):** Implementação das funções de lógica, seleção de elementos e controle de eventos.

---

## 💻 Conceitos Praticados

* Declaração e escopo de funções (`function`).
* Manipulação de strings utilizando **Template Literals** (ex: `` `game-${id}` ``).
* Utilização de seletores do ecossistema do navegador (`document.getElementById`, `querySelector`).
* Estruturas condicionais para controle de fluxo.
